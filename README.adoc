The Monkey programming language, implemented in pure C.

Monkey C, monkey do.

== Quickstart

Ensure you have the Clang compiler.

=== All platforms

----
./build.sh
----

Alternatively, install https://github.com/gittup/tup[tup] and run:

----
tup
----
